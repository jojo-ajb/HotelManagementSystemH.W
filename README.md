# 🏨 Hotel Management System

## 📌 Project Overview
This project is a Java-based Hotel Management System developed using NetBeans IDE. It is designed to manage hotel operations such as room booking, customer management, ordering system, and payment processing.

The system follows a layered architecture including:
- UI Layer
- Business Logic Layer
- Database Layer (DAO Pattern)

---

## ⚙️ Features
- Room Booking Management
- Customer Information Handling
- Food and Item Ordering System
- Extra Orders Management
- Payment Processing
- Database Integration (SQLite)
